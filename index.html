如何在GitHub和hexo基础上搭建个人博客

发表于 2024-09-04

首先是一些基础软件的下载和创建.

  1. git下载
  2. node.js下载
  3. hexo下载
  4. 创建GitHub账号和仓库

git下载

(git下载教程地址)
鼠标右键更多选项里有一个git bash here.
输入:git -version
如果有版本号,结果就是成功了.

node.js下载

(node.js官方下载地址)
在git里输入:node -v
同理如果有版本号就说明成功了

hexo下载

安装命令：npm install hexo -g
测试是否成功:hexo -v
结果为版本号就成功了.

  • 还要安装一个hexo的依赖
    命令:npm install –save hexo-deployer-git

创建GitHub账号和仓库

创建不用多说,仓库名的格式为:(username.github.io)
